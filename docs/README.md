# Abbotsford Croquet Club - Documentation & Resources

This directory contains development files, documentation, and project materials for the Abbotsford Croquet Club that are not part of the public website.

## Directory Structure

### [/city-application](./city-application/)
Application materials and guidance for offering a public recreation course through the City of Abbotsford. Includes:
- Course outline and curriculum
- Instructor qualifications documentation
- Budget planning templates
- Application checklist
- Marketing materials
- Correspondence tracking

**Purpose:** To organize all materials needed to apply for and run a city-sponsored croquet instruction course for adults and seniors.

### [/build-scripts](./build-scripts/)
Utility scripts for development and maintenance:
- `generate_favicons.sh` - Generates favicon files from logo source

### [/design-assets](./design-assets/)
Original design files and assets not used directly on the website:
- `abby-croquet-cover.png` - Cover image (9.5 MB)
- Source files for graphics

### [/promotional](./promotional/)
Promotional materials and campaign assets:
- Tournament announcements
- Event promotions
- Historical promotional content

### [/archive-files](./archive-files/)
Document archives and miscellaneous files:
- **documents/** - PDFs, tournament rules, coaching materials, invoices
- **misc/** - SVG files, Jupyter notebooks, HTML backups, experimental files

### [/email-archive](./email-archive/)
Historical email correspondence (.eml files):
- Club communications
- Order confirmations
- Society filings
- Tournament coordination

### [/reference-images](./reference-images/)
Additional images not used on the live website:
- Logo variations and drafts
- AI-generated concept images (DALL-E)
- Schedule graphics
- QR codes
- Profile images
- Experimental designs

## Organization Notes

This `/docs` folder was created to separate development and project materials from the live website files. The website files remain in the repository root to maintain compatibility with Netlify deployment.

**Website files (in root):** Public-facing content deployed to abbotsfordcroquet.ca
**Documentation files (in /docs):** Internal materials, not deployed
