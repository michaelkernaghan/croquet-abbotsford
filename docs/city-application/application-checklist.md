# City Recreation Course Application Checklist

Use this checklist to ensure all required materials and information are prepared for the application to the City of Abbotsford.

## Pre-Application Research

- [ ] Review City of Abbotsford recreation course application process
- [ ] Contact Recreation Services department for guidance
- [ ] Obtain current application forms
- [ ] Research similar course offerings in the city
- [ ] Verify insurance requirements
- [ ] Confirm facility availability at Mill Lake Park

## Course Planning

- [ ] Define course title and description
- [ ] Determine course duration and schedule
- [ ] Set class size limits
- [ ] Develop session-by-session curriculum
- [ ] Identify learning objectives
- [ ] Plan for weather contingencies
- [ ] Determine equipment needs

## Instructor Information

- [ ] Identify qualified instructor(s)
- [ ] Document croquet experience and expertise
- [ ] List relevant certifications (First Aid, CPR, coaching)
- [ ] Prepare instructor biography
- [ ] Obtain criminal record check (if required)
- [ ] Document teaching experience
- [ ] Prepare references

## Financial Planning

- [ ] Calculate course costs (instructor, materials, insurance)
- [ ] Determine participant fee structure
- [ ] Create budget breakdown
- [ ] Plan for minimum/maximum enrollment
- [ ] Consider scholarship or subsidy options
- [ ] Review city revenue sharing policies

## Documentation Required

### Core Application Materials
- [ ] Completed recreation course application form
- [ ] Detailed course outline
- [ ] Session-by-session plan
- [ ] Instructor qualifications and resume
- [ ] Course budget and fee structure
- [ ] Marketing description (for course catalog)

### Insurance & Liability
- [ ] Proof of liability insurance
- [ ] Waiver and release forms
- [ ] Safety protocols document
- [ ] Emergency procedures plan
- [ ] Incident reporting procedures

### Organization Information
- [ ] Abbotsford Croquet Club background
- [ ] Club constitution/bylaws
- [ ] Board contact information
- [ ] Proof of non-profit status (if applicable)
- [ ] Bank account information (for payments)

### Supporting Materials
- [ ] Photos of croquet courts/facilities
- [ ] Equipment inventory list
- [ ] Sample instructional materials
- [ ] Letters of support (optional)
- [ ] References from similar programs

## Marketing Materials

- [ ] Course title and tagline
- [ ] Short description (50 words)
- [ ] Full course description (200 words)
- [ ] Target audience definition
- [ ] Course highlights/benefits
- [ ] Photos for promotional use
- [ ] Social media content

## Logistics Planning

- [ ] Confirm court availability for all dates
- [ ] Identify backup dates for weather cancellations
- [ ] Plan equipment storage and transport
- [ ] Arrange for parking information
- [ ] Create participant communication plan
- [ ] Develop registration process
- [ ] Plan for first-day logistics

## Accessibility Considerations

- [ ] Document accessibility features of courts
- [ ] Plan accommodations for participants with disabilities
- [ ] Ensure wheelchair-accessible pathways
- [ ] Consider sensory accommodations
- [ ] Prepare modified equipment options
- [ ] Create inclusive participation plan

## Health & Safety

- [ ] First aid kit inventory
- [ ] Trained first aid responder identified
- [ ] Emergency contact procedures
- [ ] Weather cancellation policy
- [ ] COVID-19 protocols (if applicable)
- [ ] Allergy/medical information collection plan
- [ ] Sun safety guidelines

## Legal & Administrative

- [ ] Review city terms and conditions
- [ ] Understand cancellation policies
- [ ] Review payment and revenue sharing
- [ ] Understand city's promotional support
- [ ] Review facility use agreements
- [ ] Confirm equipment liability coverage

## Post-Application

- [ ] Submit complete application package
- [ ] Track application status
- [ ] Respond promptly to city questions
- [ ] Attend any required meetings
- [ ] Make requested revisions
- [ ] Obtain final approval
- [ ] Confirm course listing in city catalog

## Pre-Launch

- [ ] Finalize course dates and times
- [ ] Set up registration system
- [ ] Promote course through city channels
- [ ] Promote through club website/social media
- [ ] Prepare participant welcome package
- [ ] Confirm all equipment ready
- [ ] Conduct site walk-through
- [ ] Brief all instructors/volunteers

## Evaluation Planning

- [ ] Create participant feedback form
- [ ] Plan for mid-course check-in
- [ ] Develop success metrics
- [ ] Establish reporting requirements to city
- [ ] Plan for post-course follow-up
